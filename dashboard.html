Database Schema

Our PostgreSQL database is designed to handle millions of customers and billions of interactions:

Core Tables
  • clients - Your company information and API configurations
  • customers - Customer data with beauty-specific profiles
  • products - Product catalog with beauty attributes
  • orders - Order data with AI attribution tracking
Analytics Tables
  • ai_interactions - All AI interactions (questionnaires, chat, image analysis)
  • recommendations - Product recommendations and their performance
  • daily_metrics - Pre-aggregated daily analytics
  • usage_tracking - API usage and billing data

Complete API reference for both the data you need to provide and the analytics APIs we offer:

SDK & Libraries

Ready-to-use SDKs and libraries for seamless integration with SmartShopie AI:

JavaScript SDK

Client-side integration for web applications

npm install smartshopie-js-sdk
Quick Start: 
import SmartShopie from 'smartshopie-js-sdk';

const client = new SmartShopie({
  apiKey: 'your-api-key',
  clientId: 'your-client-id'
});

// Track AI interaction
client.trackInteraction({
  type: 'questionnaire',
  customerId: 'customer-123',
  data: { skinType: 'combination' }
});
Python SDK

Server-side integration for Python applications

pip install smartshopie-python
Quick Start: 
from smartshopie import SmartShopieClient

client = SmartShopieClient(
    api_key="your-api-key",
    client_id="your-client-id"
)

# Sync customer data
client.sync_customers([{
    "id": "customer-123",
    "email": "customer@example.com",
    "skin_type": "combination"
}])
PHP SDK

Integration for PHP/WordPress applications

composer require smartshopie/php-sdk
Quick Start: 
<?php
require_once 'vendor/autoload.php';

use SmartShopie\Client;

$client = new Client([
    'api_key' => 'your-api-key',
    'client_id' => 'your-client-id'
]);

// Track product recommendation
$client->trackRecommendation([
    'customer_id' => 'customer-123',
    'product_id' => 'product-456',
    'event' => 'click'
]);
React Native SDK

Mobile app integration for iOS & Android

npm install smartshopie-react-native
Quick Start: 
import SmartShopie from 'smartshopie-react-native';

// Initialize
SmartShopie.configure({
  apiKey: 'your-api-key',
  clientId: 'your-client-id'
});

// Track user interaction
SmartShopie.trackInteraction({
  type: 'image_analysis',
  customerId: 'customer-123'
});
